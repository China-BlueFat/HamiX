# 哈米记 · HamiX

> 适配小米全系 Root 设备｜本地 OCR 识别｜取件 / 取餐码 HyperOS3 上岛
> 
> 

## 📖 项目介绍

**哈米记（HamiX）**，专为 Root 后的小米 / HyperOS3 设备打造的取件 / 取餐码上岛工具。

### 名称寓意

- **中文名：哈米记**

    - 哈：四川话里的“哈儿”的意思；

    - 米：小米；

    - 记：取件 / 取餐码信息**记录与展示**。

- **英文名：HamiX**

    - `Hami` 取自「哈米」拼音；

    - 后缀 `X` 代表扩展、增强、进阶能力，持续迭代新增功能。

## ✨ 核心功能

1. 本地 OCR 识别，识别快递取件码、餐饮取餐码；

2. 识别结果自动记录归档，方便随时查看；

3. 支持 HyperOS3 通知胶囊「上岛」展示关键码单信息；

4. 全局快捷操作：授予 Root 权限后，**双击音量下键**一键唤起识别；

5. 高度自定义：支持手动编辑品牌规则、替换餐饮品牌图标。

## 📂 配置文件目录

自定义规则、识别配置、品牌图标资源存放路径：

```Plain Text
/storage/emulated/0/Android/data/com.zayne.hamix/files/
```

> 提示：项目内置规则暂未完全覆盖全品牌，有需求可自行手动补充配置。
> 
> 

## 🛠 技术栈

- 全量界面基于 **MiuiX** 框架开发，贴合小米系统原生视觉风格；

- 识别方案：纯**本地 OCR**，无网络请求、无云端上传，保护本地隐私；

- 后续规划：迭代接入 AI 识别能力，提升复杂场景识别准确率。

## ⚠️ 使用前提

- 设备：小米 / 红米手机，HyperOS3；

- 权限：设备已获取 **Root 权限**；

- 权限授予：需为软件授予 Root 权限，方可使用双击音量下快捷识别。
